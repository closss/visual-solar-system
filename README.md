# 口袋宇宙实验台

基于 `Three.js + TypeScript + Vite` 的网页图像增强演示：在预设场景图上叠加透明 3D 微缩宇宙装置，支持锚点放置、太阳-地球-月亮层级动画、样条路径探测器、科学/电影模式、时间倍率、截图保存和移动端质量降级。

## 快速启动

```bash
npm install
npm run dev
```

打开 `http://127.0.0.1:5173/`。

## 已落地功能

- 两个预设场景：桌面实验台、校园夜幕。
- 点击画面放置微缩宇宙，支持重置锚点和上传自定义场景图。
- 三体层级关系：太阳、地球、月球。
- 探测器沿 `CatmullRomCurve3` 样条路径飞行。
- `Science / Cinematic` 双模式。
- 时间倍率滑杆、增强前/后对比、整页截图保存、FPS 监控。
- 移动端自动切换 `safe` 质量档位。

## 验证

```bash

```
